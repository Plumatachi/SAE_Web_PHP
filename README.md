# SAE_Web_PHP

## Base de données
Pour crée la Base de données vous devais faire les commandes:

Sur Windows
```shell
php .\Data\create-database

php .\Data\create-table

php .\Data\creationbd.php load-data
```

Sur linux
```bash
php ./Data/create-database

php ./Data/create-table

php ./Data/creationbd.php load-data
```
